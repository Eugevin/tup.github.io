You have text? Up it to the new level!

In search of an ideal solution for processing and colorful appearance of text, I created this library. You can see a brief description of the principles of its work in the video
Test content
TypeWriter
span {opacity: 0} tup__item {opacity: 1}
InOut
span {display: inline-block; transform: scale(2); opacity: 0; transition: .7s;} tup__item {transform: scale(1); opacity: 1;}
FadeIn
position: relative; left: -20px; opacity: 0; transition: .7s; .tup__item { left: 0; opacity: 1; }
Rotate
span {display: inline-block; transform: scale(2); opacity: 0; transition: .7s;} tup__item {transform: scale(1); opacity: 1;} @keyframes rotateAnimation { 0% { background: #000; color: #fff; } 50% { background: #fff; color: #000; } 100% { background: #000; color: #fff; } }
You just need to set the initial span value and the final tup__item value, as well as add animation as desired. The number of options and methods of use is limited only by your imagination!

How to use?
  • 1) Connect tup.js before body end tag and initialize it
  • 2) Add tup class in your classes
    My text
  • 3) Add additional parameters if you want
    My text
  • 4) Animate your span and tup__item (you can use @keyframes if you wish)
    span {
  opacity: 0;
}

.tup__item {
  opacity: 1;
  animation: anim .3s ease infinite;
}
